# Simple Banking System (Console App)

A basic Python console application that performs simple banking operations like deposit, withdrawal, and balance checking.  
This project demonstrates **Python file handling** using JSON storage.
## Features
- Deposit money  
- Withdraw money (with validation)  
- Check current balance  
- Balance stored in a JSON file (`account_data.json`)  
## Files Included
- `banking_system.py` — Main script  
- `account_data.json` — Auto-created to store balance  
- README.md — Instructions and details  
## How to Run
### 1. Install Python (if not installed)
Download from: https://www.python.org/downloads/
